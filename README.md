# EJERCICIO-CLASE
El algoritmo de Divide y Vencerás reduce la complejidad de O(n^2) a O(n \log n) porque evita comparar todos los pares de puntos, que es lo que hace la fuerza bruta. Lo logra al dividir el problema en dos mitades y resolverlas recursivamente (2T(n/2)). El paso crucial es la fase de combinación, donde solo se analizan los puntos dentro de una "banda" estrecha cerca de la línea divisoria. Gracias a una propiedad geométrica, en esta banda solo es necesario comparar cada punto con un número constante (máximo 7) de sus vecinos ordenados por y, haciendo que la combinación solo tome un tiempo lineal, O(n). Al sumar esto, la recurrencia resultante es T(n) = 2T(n/2) + O(n), lo que se resuelve como \mathbf{O(n \log n)}.
